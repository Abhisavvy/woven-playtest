# WOVEN — playtest build

A deterministic touch-drag puzzle: 2–4 coloured threads must partition every cell
of the grid — fill everything, never cross.

**Play it:** https://abhisavvy.github.io/woven-playtest/

Single self-contained `index.html` (no server, no build, no account). Works on
mobile and desktop; progress and coins are saved per-browser (localStorage).

60 levels across 6 tiers — core routing, gates, weave/overlap, logic gates,
3–4 threads, and the blend mechanic — with earlier mechanics recurring in later
tiers along a rising difficulty curve. This is a throwaway prototype for
playtesting puzzle progression and difficulty feedback.
